# TopologyGenerator — 数字拓扑图生成器

一个基于 **OpenGL** 的 3D 拓扑结构可视化工具，支持实时生成和交互式探索多种拓扑结构，包括 **二维拓扑图、球体结构与圆环结构**。  
用户可通过直观的控制面板调整参数，实时观察结构变化，并支持保存高质量截图。

> 🎯 **无需额外安装依赖、无需手动配置环境、无需修改代码**  
> **解压项目文件 → 打开 `.sln` → 编译 → 运行**

---

## 01 快速开始

### 方法一：直接打开exe文件
- 打开TopologyGenerator.exe

### 方法二：下载并解压项目

1. 下载本仓库源码（或 Release 中的压缩包）

## 02 关于第三方依赖（重要说明）

由于本项目依赖的第三方库（如 **glfw / glad / glm / imgui / stb**）
**文件数量较多，直接上传到 GitHub 会触发文件数量与大小限制**，因此本仓库采用如下方式处理依赖项：

> 📦 **所有第三方依赖以 `dependencies.zip` 的形式提供**

### 使用方式

1. **请在 `TopologyGenerator/` 目录下解压 `dependencies.zip`**
2. 解压后应得到如下结构：

```text
TopologyGenerator/
├── dependencies/
│   ├── glfw/
│   ├── glad/
│   ├── glm/
│   ├── imgui/
│   └── stb/
```

3. 保持目录结构不变，即可正常编译运行项目

> 💡 解压完成后，项目将自动通过 `Dependencies.props` 引入所有依赖，
> **无需手动修改 Visual Studio 的包含目录或库目录配置。**

## 03 功能特点
- 多拓扑结构支持：提供 2D 拓扑图、3D 球体和 3D 圆环三种基本结构
- 实时参数调整：通过交互式控制面板修改结构参数，即时更新可视化效果
- 美化功能：开启 "Beautify" 选项可添加内部装饰结构，提升视觉复杂度
- 自动旋转：支持启用 / 关闭自动旋转，便于全方位观察 3D 模型
- 交互控制：右键拖拽旋转模型，鼠标滚轮缩放视图
- 数据统计：实时显示当前结构的顶点数量和边数量
- 截图功能：一键保存当前视图为 PNG 图片，自动存储到screenshots文件夹中

> ✅ 已完成所有`基础项`和`加分项`

## 05 参数说明
### 2D 拓扑图
- 子结构数量 (3-100)：控制 2D 拓扑图的边数
- 每个子结构的点数 (1-50)：仅在美化模式下可见，控制内部装饰点数量
### 球体
- 子结构数量 (3-100)：控制球体的扇区数量
- 每个子结构的点数 (1-50)：仅在美化模式下可见，控制螺旋装饰的点数
- 高度层数 (2-50)：控制球体的垂直分层数量
### 圆环
- 主环分段数 (4-100)：控制圆环主体的分段数量
- 管环分段数 (3-50)：控制圆环管道的分段数量
- 主环半径 (0.5-2.0)：调整圆环主体的半径大小
- 管环半径 (0.1-1.0)：调整圆环管道的半径大小



## 06 代码结构
### Application：应用程序主控制器
- 管理窗口创建、渲染循环和用户输入
- 集成 ImGui 控制面板，处理参数调整逻辑
- 负责 OpenGL 渲染管线和缓冲区管理
- 实现截图保存功能，自动创建screenshots目录

### TopologyGraph：拓扑结构生成核心
- 根据参数生成不同类型的拓扑结构（2D 拓扑图、球体、圆环）
- 管理顶点和索引数据的存储与生成
- 提供美化模式下的额外结构生成逻辑

### Parameters：参数配置结构体
- 存储所有可调整的拓扑结构参数
- 包含结构类型选择、美化开关和自动旋转开关
